# 🎨 极简纸质风格设计说明

## 设计理念

参考数据可视化的极简主义风格，采用纸质感背景配合简洁的线条和点，打造专业、优雅的现代界面。

---

## 🎨 配色方案

### 核心颜色

| 变量名 | 色值 | 用途 | 灵感来源 |
|--------|------|------|---------|
| `--paper-bg` | `#F2F1ED` | 页面背景 | 纸张/图表背景色 |
| `--paper-light` | `#FAFAF8` | 容器背景 | 白纸质感 |
| `--text-dark` | `#2D2D2D` | 主文字 | 图表数据点/线条 |
| `--text-gray` | `#6B6B6B` | 次要文字 | 图表标注 |
| `--text-light` | `#9B9B9B` | 辅助文字 | 浅色提示 |
| `--line-color` | `#D4D3CE` | 边框/分隔线 | 图表网格线 |
| `--line-dark` | `#8B8B8B` | 强调线条 | 图表参考线 |
| `--accent-orange` | `#E67E50` | 强调色 | 图表高亮色 |
| `--dot-color` | `#2D2D2D` | 数据点 | 图表数据点 |

### 配色特点

1. **纸质感背景** - `#F2F1ED` 模拟真实纸张的温暖米色调
2. **高对比度文字** - 深灰色文字在浅色背景上清晰可读
3. **极简线条** - 使用细线条（1px）代替粗重边框
4. **橙色点缀** - 用于按钮和链接的强调色

## 📐 设计元素

### 1. 排版

**标题**
- 左对齐（替代居中）
- 底部黑色粗线条分隔
- 去除渐变效果
- 字重 600

**副标题**
- 左对齐
- 浅灰色 `#6B6B6B`
- 字号 0.95em

### 2. 输入框

```css
border: 1px solid #D4D3CE  /* 细边框 */
border-radius: 2px          /* 最小圆角 */
padding: 12px 16px          /* 适中内边距 */
```

**特点**：
- 细边框（1px）
- 最小圆角（2px）
- 聚焦时边框变为深灰色

### 3. 按钮

**主按钮（深色）**
```css
background: #2D2D2D
border: 1px solid #2D2D2D
color: #FAFAF8
```

**次要按钮（线框）**
```css
background: transparent
border: 1px solid #8B8B8B
color: #2D2D2D
```

**危险按钮**
```css
background: transparent
border: 1px solid #C84B31
color: #C84B31
```

**设计特点**：
- 无阴影
- 细边框
- 扁平化
- 悬停时简单的背景色变化

### 4. 表格

**表头**
```css
background: #F2F1ED  /* 纸质背景 */
color: #6B6B6B       /* 灰色文字 */
text-transform: uppercase
letter-spacing: 0.05em
```

**表格行**
```css
border-bottom: 1px solid #D4D3CE  /* 细线分隔 */
hover: background #FAFAF8          /* 悬停浅色 */
```

**特点**：
- 表头小写转大写
- 细线分隔
- 无粗重边框
- 最后一行无底边框

### 5. 卡片/容器

```css
background: white
border: 1px solid #D4D3CE
border-radius: 2px
padding: 24px
```

**特点**：
- 白色背景
- 细边框
- 最小圆角
- 无阴影

### 6. 统计卡片

```css
background: white
border: 1px solid #D4D3CE
```

**数字**
```css
font-size: 2.5em
font-weight: 600
letter-spacing: -0.02em
font-feature-settings: 'tnum'  /* 表格数字 */
```

**标签**
```css
text-transform: uppercase
letter-spacing: 0.05em
color: #6B6B6B
```

## ✨ 细节优化

### 1. 字体设置

```css
font-feature-settings: 'tnum'  /* 等宽数字 */
letter-spacing: -0.02em        /* 紧凑字距 */
```

### 2. 线条使用

- 所有边框使用 1px
- 分隔线使用 `border-bottom: 1px solid #D4D3CE`
- 强调线使用 2px

### 3. 圆角统一

```css
--border-radius: 2px  /* 全局最小圆角 */
```

### 4. 间距系统

基于 8px 倍数：
- 8px, 12px, 16px, 24px, 32px, 48px

### 5. 去除装饰

- ❌ 移除所有 emoji
- ❌ 移除渐变色
- ❌ 移除阴影效果
- ❌ 移除圆润的圆角
- ✅ 使用简洁的线条和点
- ✅ 使用扁平化设计

## 🔧 如何调整

### 更改主色调

只需修改 `--accent-orange`:
```css
--accent-orange: #E67E50;  /* 橙色 */
--accent-orange: #3B82F6;  /* 蓝色 */
--accent-orange: #10B981;  /* 绿色 */
```

### 更改背景色

调整 `--paper-bg`:
```css
--paper-bg: #F2F1ED;  /* 米色 */
--paper-bg: #F5F5F0;  /* 浅黄 */
--paper-bg: #FAFAFA;  /* 浅灰 */
```

### 更改线条粗细

```css
border: 1px solid var(--line-color);  /* 改为 2px */
```
